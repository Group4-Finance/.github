## Group4-Finance

專題主題：
# 💡 買賣不FOMO：ETF買賣決策助理

ETF 是許多上班族偏好的穩健投資選擇，但在資訊爆炸、時間不足的生活下常導致決策焦慮，難以做決定。

## 🎯 專案目標

本專案旨在打造一套「ETF買賣決策助理」，讓你在繁忙生活中也能理性判斷買點，避免 FOMO（錯失恐懼）或觀望過久。
\
本系統從三大面向提供即時分析與建議：
- 📉 折溢價分析：評估買入價值是否合理  
- 😨 市場情緒評估：參考 VIX 與 Fear & Greed 指數掌握風險氛圍  
- 📣 新聞與社群輿情：結合情緒分析，判斷大眾觀點趨勢
